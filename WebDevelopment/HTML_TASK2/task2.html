How to design a favicon using favicon.cc

Following are the steps to create a favicon using favicon.cc

  • Open the website favicon.cc
  • select option create new favicon
  • pick the colour of your choice from the right color picker panel
  • draw the pattern of your desired style with desired colors
  • if you want an animated favicon please find the option under the drawing area(optional)
  • select the type of animation you wish to use
  • Preview the favicon under the icon editor and
  • click on download favicon and you can find it in your local system
